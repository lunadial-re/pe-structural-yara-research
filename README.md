# Методы ускоренного первичного статического анализа вредоносного ПО на основе структурных аномалий формата PE с использованием правил YARA

## Описание
Данный репозиторий представляет собой исследовательскую работу, посвященную разработке и оценке методов ускоренного первичного статического анализа малвари на основе выявления типовых структурных аномалий формата Portable Executable с использованием правил YARA.
Основная идея исследования заключается в том, что значительная часть образцов может демонстрировать устойчивые отклонения от типичного легитимного PE-профиля, которые могут быть обнаружены на раннем этапе анализа без выполнения кода, привязки к хешам, строкам и сетевым индикаторам.

## Актуальность
Полноценный ручной анализ каждого образца является трудоемким и плохо масштабируемым.
На практике, как мне кажется, наибольшую ценность в первичном анализе могут представлять методы, которые позволят:
- быстро отсеять очевидно нерелевантные или малоприоритетные образцы
- выявить подозрительные файлы до этапа динамического анализа
- формировать инварианты\эвристики, в меньшей степени подверженные быстрой модификации со стороны разработчика малвари
- (в перспективе) могут быть интегрированы в SOAR\EDR\XDR-системы

В таком случае использование структурных характеристик PE-файлов в сочетании с правилами YARA будет рассматриваться в данной работе как один из возможных подходов к решению этих задач

## Цель исследования
Разработка и экспериментальная оценканабора YARA-правил, предназначенных для ускоренного первичного статического анализа малвари на основе структурных аномалий формата PE

## Задачи исследования
- анализ типовых структурных аномалий PE-файлов, характерных для вредоносного ПО
- формализация выявленных аномалий в виде правил YARA
- (возможно) классификация индикаторов в соответствии с принципами устойчивости ("Pyramid of Pain"-like методология)
- оценка применимости разработанных правил
- анализ ограничений подхода и потенциальных источников FP

## Область применения
- первичный триаж образцов
- предвариетльная фильтрация PE-файлов
- вспомогательный инструмент для анализа ВПО
- (в перспективе) полноценная тулза или тулкит для первичного статического анализа ВПО
- (в перспективе) использование в качестве вспомогательного сигнала в системах XDR\EDR

## Структура репозитория (предполагаемая)
- исследовательные материалы и описание методологии
- наборы YARA-правил, сгруппированые по типам структурных аномалий
- материалы по тестированию и оценке результатов

## Статус проекта
Проект в активной стадии разработки. Репозиторий будет регулярно обновляться.
